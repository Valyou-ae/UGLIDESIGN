# STYLE: Gothic Architecture

## Overview
Medieval European architecture characterized by pointed arches, flying buttresses, and intricate stonework.

## Key Characteristics
- Pointed arches
- Ribbed vaults
- Flying buttresses
- Rose windows
- Vertical emphasis
- Intricate stone tracery
- Stained glass

## Essential Keywords
`Gothic architecture`, `medieval cathedral`, `pointed arches`, `flying buttresses`, `Gothic revival`

## Best Use Cases
- Cathedrals and churches
- Fantasy castles
- Dark atmospheric scenes
- Historical architecture
