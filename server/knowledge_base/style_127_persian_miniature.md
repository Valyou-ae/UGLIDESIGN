# STYLE: Persian Miniature

## Overview
Intricate Islamic art tradition featuring detailed flat compositions with rich colors and gold leaf.

## Key Characteristics
- Flat perspective (no depth)
- Intricate patterns and details
- Rich jewel tones
- Gold leaf accents
- Multiple scenes in one image
- Decorative borders
- No shadows

## Essential Keywords
`Persian miniature`, `Islamic miniature painting`, `Mughal miniature`, `Ottoman miniature`
