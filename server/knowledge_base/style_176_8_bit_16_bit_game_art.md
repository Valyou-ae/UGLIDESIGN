# 176. 8-BIT/16-BIT GAME ART

### **Core Characteristics:**
- **Technical Limitation**: Hardware constraints create aesthetic
- **Pixel Perfect**: Every pixel hand-placed and meaningful
- **Color Palette**: Limited colors (NES: 54 colors, SNES: 32,768 possible)
- **Tile-Based**: Repeating background tiles, sprite-based characters
- **Nostalgia Factor**: 1980s-1990s golden age of gaming

### **8-Bit Specifications (NES Era, 1983-1994):**

**Technical Constraints:**
```
Resolution:
- Screen: 256 x 240 pixels
- Sprites: 8x8 or 8x16 pixels
- Background tiles: 8x8 pixels

Color Limitations:
- Total palette: 54 colors available
- On-screen simultaneously: 25 colors maximum
- Per sprite: 3 colors + transparency
- Per background tile: 3 colors + background color

Sprite Limits:
- 64 sprites total on screen
- 8 sprites per horizontal line
- Sprite flickering when exceeded
```

**Visual DNA:**
```
Character Design:
- Large heads (readability)
- Minimal details
- Strong silhouettes
- Exaggerated proportions
- Limited animation frames (3-4 for walk cycle)

Background Design:
- Repeating tile patterns
- Dithering for color mixing
- Simple shapes
- Strong color contrast
- Black outlines common
```

**Iconic Color Palettes:**
```
NES Palette Groups:
- Bright primaries (Mario)
- Dark grays/blues (Mega Man)
- Earth tones (Zelda)
- Pastel limited (Kirby)

Common Techniques:
- Black outlines for definition
- Dithering (checkerboard patterns) for shading
- Single color gradients
- High contrast for readability
```

### **16-Bit Specifications (SNES/Genesis Era, 1988-1999):**

**Technical Advancement:**
```
Resolution:
- Screen: 256 x 224 to 512 x 448 (various modes)
- Sprite sizes: 8x8 to 64x64 pixels
- More background layers (parallax scrolling)

Color Explosion:
- SNES: 32,768 colors possible, 256 on-screen
- Genesis: 512 colors possible, 64 on-screen
- Per sprite: 15 colors + transparency
- Gradient shading now common

Sprite Improvements:
- 128 sprites on screen
- Larger sprite sizes
- More animation frames (6-8+ walk cycle)
- Rotation and scaling (Mode 7 on SNES)
```

**Visual Evolution:**
```
Character Design:
- More detailed anatomy
- Realistic proportions possible
- Detailed facial features
- Complex animations
- Clothing texture visible

Background Design:
- Parallax scrolling (depth layers)
- Gradient skies
- Detailed textures
- Less tiling repetition
- Atmospheric effects (fog, transparency)
```
