# 170. STREETWEAR

### **Core Characteristics:**
- **Origin**: Hip-hop, skate, surf culture fusion
- **Philosophy**: Limited drops, hype culture, status through exclusivity
- **Aesthetic**: Casual luxury, logo-driven, collectible
- **Innovation**: Direct-to-consumer, social media marketing

### **The Streetwear DNA:**

**Foundational Elements:**
```
The Formula:
- Graphic t-shirts (bold logos, artwork)
- Hoodies (oversized, branded)
- Sneakers (limited edition, collectible)
- Cargo pants or joggers
- Caps (fitted, snapback, dad hat)
- Accessories (bags, socks, etc. with branding)

Key Principles:
- Limited quantities (scarcity)
- Collaboration culture
- Brand loyalty/tribalism
- Resale value consideration
- Drop culture (specific release times)
- Queuing/camping for releases
```

```
Condition Terms:
- Deadstock (DS): Brand new, unworn
- VNDS: Very near deadstock
- Beaters: Worn, for daily use
```
