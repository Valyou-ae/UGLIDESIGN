# 173. HISTORICAL COSTUME

### **Core Principles:**
- **Accuracy**: Period-appropriate construction, materials, silhouettes
- **Context**: Social class, occupation, geography, occasion
- **Research**: Primary sources, museum pieces, artwork, written records
- **Purpose**: Theater, film, reenactment, museums, education

### **Research Methodology:**

**Primary Sources:**
```
Visual Documentation:
- Period paintings and portraits
- Photographs (post-1840s)
- Fashion plates and illustrations
- Sculpture and statuary
- Surviving garments (museums)
- Archaeological textiles

Written Sources:
- Tailoring manuals
- Fashion magazines
- Personal letters/diaries
- Legal sumptuary laws
- Trade records
- Literature descriptions
```

**Museum Resources:**
```
Major Collections:
- Metropolitan Museum Costume Institute (NYC)
- Victoria & Albert Museum (London)
- Kyoto Costume Institute (Japan)
- Musée des Arts Décoratifs (Paris)
- Fashion Institute of Technology Museum
- Colonial Williamsburg

Study Approach:
- Examine construction methods
- Note fabric choices
- Understand layering systems
- Document fastening techniques
- Analyze wear patterns
```

### **Period Breakdown:**

**Ancient World (3000 BCE - 476 CE):**
```
Egyptian:
- Linen (primary fabric)
- Pleating techniques
- White dominant color
- Simple draping
- Gold jewelry
- Kohl eye makeup
- Shaved heads with wigs

Greek & Roman:
- Draped garments (chiton, toga)
- Wool and linen
- Purple for status (Tyrian purple)
- Brooches (fibulae) for fastening
- Sandals
- Laurel wreaths
- Gender distinctions in draping
```

**Medieval (476 - 1450 CE):**
```
Early Medieval:
- Tunics (men and women)
- Simple construction
- Natural dyes
- Wool primary fabric
- Linen undergarments
- Cross-garter leg wrappings

High Medieval:
- Surcoats and cotes
- Longer, fuller garments
- Heraldic symbols
- Pointed shoes (poulaines)
- Wimples and veils (women)
- Chainmail and armor (knights)

Late Medieval:
- Fitted garments emerge
- Buttons appear
- Houppelande (voluminous gown)
- Hennin (tall conical hat)
- Dagging (decorative edges)
- Rich brocades
```

**Renaissance (1450 - 1600):**
```
Italian Renaissance:
- Rich velvets and brocades
- Slashed sleeves
- Square necklines
- Chemise visible
- Jeweled accessories
- Complex embroidery

Tudor England:
- Farthingale (hoop skirt)
- Ruff collars (increasing size)
- Blackwork embroidery
- Paned and slashed fabric
- Codpiece (men)
- French hood, gable hood

Elizabethan:
- Extreme ruffs
- Cartwheel farthingale
- Bombast padding
- Stomacher (decorative center)
- Rich embroidery and jewels
- Pale makeup, red hair fashionable
```

**17th Century:**
```
Early (1600-1650):
- Falling collar replaces ruff
- Natural waistline returns
- Lace dominance
- Van Dyck style
- Cavalier fashion
- Petticoat breeches (men)

Late (1650-1700):
- Mantua gown (women)
- Fontange (high headdress)
- Justaucorps coat (men)
- Cravat and jabots
- Periwig (full wigs, men)
- High heels introduced
```

**18th Century:**
```
Rococo (1715-1789):
- Pannier skirts (side hoops)
- Pastel colors
- Delicate florals
- Robe à la française
- Stomacher bodice
- Powdered wigs (both sexes)
- Beauty patches

Men's Fashion:
- Three-piece suit emerges
- Coat, waistcoat, breeches
- Silk and brocade
- Wigs (various styles)
- Buckled shoes
- Tricorn hat

Revolutionary Era (1789-1799):
- Simpler silhouettes
- Natural fabrics (cotton)
- Higher waistlines emerging
- Sans-culottes (working class)
- Phrygian cap (revolutionary symbol)
```

**19th Century:**
(See separate Victorian, Edwardian entries for detail)
```
Empire/Regency (1800-1820):
- High waistline
- Flowing skirts
- Sheer fabrics
- Spencer jacket
- Bonnet hats

Romantic (1820-1840):
- Fuller sleeves (gigot)
- Wider skirts
- Natural waistline returns
- Elaborate hairstyles

Victorian (1837-1901):
- Crinoline, bustle eras
- Complex undergarments
- Increasing ornamentation

Edwardian (1901-1914):
- S-bend corset
- Gibson Girl
- Lighter fabrics
```

### **Construction Techniques by Era:**

**Pre-Industrial:**
```
Hand Sewing:
- All garments hand-sewn
- Running stitch, backstitch
- Whipstitch for seams
- Buttonhole stitch
- Embroidery stitches

Pattern Development:
- Flat pattern construction
- Geometric shapes
- Minimal cutting
- Fabric conservation
- No paper patterns (until 1860s)
```

**Industrial Revolution Impact:**
```
Sewing Machine (1850s+):
- Faster production
- More complex seaming
- Decorative stitching
- Wider availability

Standardization:
- Pattern companies (Butterick, McCall's)
- Sizing systems
- Mass production
- Ready-to-wear
```

### **Fabric Authenticity:**

**Natural Fibers by Period:**
```
Ancient - Medieval:
- Linen (most common)
- Wool (all classes)
- Silk (wealthy only)
- Cotton (limited, expensive)

Renaissance - 18th C:
- Silk varieties expand
- Velvet (cut pile)
- Brocade (woven patterns)
- Damask
- Taffeta
- Cotton more available

19th Century:
- Industrial textile production
- Chemical dyes (1856+)
- Wider fabric widths
- More pattern options
```

**Synthetic Warning:**
```
Avoid for Historical Accuracy:
- Polyester (invented 1941)
- Nylon (1935)
- Spandex/Lycra (1958)
- Acrylic (1950)
- Modern elastic

Exception:
- Hidden structural elements acceptable
- Budget/accessibility compromises
- Clearly labeled as reproduction
```

### **Undergarments by Era:**

**Foundation Garments:**
```
Medieval:
- Chemise/shirt (linen)
- Braies (men's underwear)
- Hose (leg coverings)

Renaissance:
- Shift (women)
- Corset emerges
- Farthingale (hoop support)
- Stockings

18th Century:
- Stays (corset)
- Panniers or pocket hoops
- Multiple petticoats
- Chemise
- Stockings (garters)

Victorian:
- Corset (heavily boned)
- Chemise
- Drawers
- Crinoline or bustle
- Multiple petticoats
- Stockings

Edwardian:
- S-bend corset
- Combination undergarment
- Petticoat
- Bust improver
- Stockings
```

### **Social Class Indicators:**

**Fabric & Color:**
```
Wealthy:
- Silk, velvet, fine wool
- Purple, scarlet (expensive dyes)
- Gold thread embroidery
- Fur trim (ermine, sable)

Middle Class:
- Wool, cotton
- Moderate colors
- Less ornamentation
- Practical

Working Class:
- Coarse wool, linen
- Natural colors
- Minimal decoration
- Durable, practical
```

**Construction Quality:**
```
High Status:
- Hand-finished seams
- Custom fit
- Intricate details
- Latest fashion

Lower Status:
- Ready-made or simple construction
- Practical cuts
- Outdated styles
- Mended/patched acceptable
```

### **Costume for Performance:**

**Theater Considerations:**
```
Modifications:
- Exaggeration for visibility
- Stronger construction (movement)
- Quick-change mechanisms
- Washability
- Budget constraints

Historical Accuracy vs. Stage:
- Silhouette accuracy priority
- Details can be suggested
- Color for stage lighting
- Durability essential
```

**Film/TV Standards:**
```
High Accuracy Required:
- Close-up scrutiny
- HD/4K resolution
- Historical consultants
- Museum-quality replicas

Practical Modifications:
- Hidden zippers (convenience)
- Modern undergarments (comfort)
- Machine sewing (speed)
- Synthetic linings (durability)
```

### **Common Mistakes to Avoid:**

```
❌ Anachronisms:
- Wrong period undergarments
- Modern closure methods visible
- Synthetic fabrics
- Incorrect hair/makeup
- Wrong jewelry style

❌ Fantasy Elements:
- Mixing multiple eras
- Excessive embellishment
- Impractical construction
- Inaccurate color combinations

✓ Accuracy Priorities:
- Correct silhouette (most important)
- Period-appropriate fabric types
- Authentic construction visible
- Proper undergarments
- Research-based details
```
