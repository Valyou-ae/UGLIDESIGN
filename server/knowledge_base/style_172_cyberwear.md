# 172. CYBERWEAR

### **Core Characteristics:**
- **Aesthetic**: Futuristic, tech-integrated, dystopian/utopian fusion
- **Materials**: Technical fabrics, LED, reflective, waterproof
- **Philosophy**: Form follows function, urban survival, tech-positive
- **Influence**: Cyberpunk, techwear, sci-fi, tactical

### **Cyberwear DNA:**

**Foundational Elements:**
```
Techwear Base:
- All-black or monochromatic
- Technical fabrics (Gore-Tex, etc.)
- Multiple pockets and storage
- Modular attachment systems
- Water resistance
- Breathable yet protective
- Urban utility focus

Cyberpunk Influence:
- Neon accents
- LED integration
- Reflective materials
- Asymmetric cuts
- Dystopian aesthetic
- Corporate logos satirized
- Street samurai vibe
```

### **Silhouette & Construction:**

**The Techwear Silhouette:**
```
Upper Body:
- Technical shells (hard or soft)
- Multiple layers (modular)
- High collars
- Adjustable hoods
- Articulated sleeves
- Chest rigs/harnesses
- Utility vests

Lower Body:
- Tapered cargo pants
- Articulated knees
- Ankle zippers
- Multiple pockets
- Water-resistant
- Gusseted crotch
- Drawstring hems

Footwear:
- High-top sneakers
- Technical boots
- Gore-Tex lined
- Aggressive tread
- Futuristic silhouette
- Often all-black
```

**Layering System:**
```
Layer 1 (Base):
- Compression shirt or base layer
- Moisture-wicking
- Form-fitting

Layer 2 (Mid):
- Technical hoodie or shirt
- Fleece or insulation
- Modular attachments

Layer 3 (Shell):
- Waterproof jacket
- Hard shell
- Multiple pockets
- Adjustable features

Accessories Layer:
- Chest rig
- Utility belt
- Bags and pouches
- Holsters
```

**Performance Fabrics:**
```
Gore-Tex:

- Various weight options
- Trusted protection

- Insulation technology

- Multiple performance levels

Cordura:

- Military-grade
- Long-lasting

Schoeller:

- Water-resistant
- Premium quality

Outlier Fabrics:

- Performance merino
- Ultra-breathable

```

```

LED Integration:
```
