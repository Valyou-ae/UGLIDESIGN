# 184. JRPG STYLE

### **Core Characteristics:**
- **Anime Influence**: Character designs from anime/manga
- **Turn-Based Combat**: Menu-driven battle systems
- **Fantasy Settings**: Medieval fantasy or sci-fi
- **Epic Narratives**: Save the world, chosen heroes
- **Detailed Sprites**: 16/32-bit era pixel art or modern HD 2D

### **Visual DNA:**

**Character Design:**
```
Anime Aesthetic:
- Large expressive eyes
- Colorful hair (unrealistic colors common)
- Spiky hair (protagonists)
- Slim, attractive proportions
- Elaborate costumes

Archetypes:
- Hero: Spiky hair, sword, determined
- Mage: Robes, staff, wise or mysterious
- Fighter: Muscular, heavy armor
- Thief/Rogue: Agile, light armor, daggers
- Healer: Gentle, white/light colors
```

**Sprite Era (16/32-Bit):**
```
Super Nintendo (1990-1996):
- 16-bit sprites
- Detailed pixel art
- Chrono Trigger: ~400 poly count equivalent in pixels
- Final Fantasy VI: Expressive character sprites
- Secret of Mana: Bright, colorful

PlayStation Era (1997-2000):
- Pre-rendered backgrounds
- 3D character models (low poly)
- Final Fantasy VII-IX: Chibi field models, detailed battle models
- Fixed camera angles
- Rich, painterly backgrounds

Modern HD-2D (2018+):
- Pixel art characters
- 3D backgrounds with tilt-shift effect
- Octopath Traveler innovation
- Best of retro and modern
```

### **Combat Visual Design:**

**Battle UI:**
```
Essential Elements:
- Character portraits
- HP/MP bars
- ATB (Active Time Battle) gauges
- Command menus (Fight, Magic, Item, Defend)
- Enemy sprites/models
- Background (often static or looping)
```

**Attack Animations:**
```
Physical Attacks:
- Character rushes forward (sprite movement)
- Weapon strike animation
- Hit effect (flash, impact)
- Damage numbers pop up
- Return to position

Magic Spells:
- Casting animation (character glows, gesture)
- Spell effect (fire, ice, lightning)
- Screen flash, particle effects
- Sound effect sync
- Damage/healing numbers
```

### **Specific JRPG Aesthetics:**

**Final Fantasy Series:**
```
Hallmarks:
- Crystals (recurring motif)
- Chocobos (yellow bird mounts)
- Moogles (cute creatures, kupo!)
- Cid (character name in every game)
- Airships (ornate, fantastical)
- Epic summons (Bahamut, Ifrit, Shiva)
```

**Dragon Quest Series:**
```
Hallmarks:
- Akira Toriyama art (Dragon Ball creator)
- Slime mascot (blue, cute)
- Medieval fantasy (knights, castles)
- First-person battles (early games)
- Bright, cheerful aesthetic
```

**Persona Series:**
```
Modern JRPG:
- Contemporary Japan setting
- High school life + dungeon crawling
- Stylish UI (best in class)
- Social sim + combat
- Anime aesthetic (sharp, bold)
```
