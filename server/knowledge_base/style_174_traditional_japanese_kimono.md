# 174. TRADITIONAL JAPANESE (KIMONO)

### **Core Characteristics:**
- **Philosophy**: Harmony, seasonality, social meaning, wearable art
- **Construction**: T-shaped, straight seams, rectangular fabric
- **Meaning**: Every element communicates (age, status, occasion, season)
- **Art Form**: Centuries of refinement, living cultural heritage

### **Kimono Fundamentals:**

**Basic Structure:**
```
Pattern Pieces:
- No curves, only straight seams
- Minimal fabric waste
- Can be disassembled and reassembled
- One size pattern, adjusted by folding/wrapping

Components:
- Hon (main body panels)
- Sode (sleeves)
- Eri (collar)
- Okumi (front inner panel)
- Maemigoro (front main panel)
- Ushiromigoro (back panel)

Measurements:
- Tan (Âèç): Full bolt of fabric (12-13 meters)
- Width: 36-38 cm (traditional)
- Final garment: Floor-length, wrapped and belted
```

**Wearing Layers:**
```
Full Traditional:
1. Hadajuban (undershirt)
2. Susoyoke (under-kimono slip)
3. Nagajuban (under-kimono, collar visible)
4. Kimono (outer garment)
5. Datejime (under-belt, 2-3 layers)
6. Obi (decorative sash, 4m long)
7. Obijime (obi cord)
8. Obiage (obi scarf)

Modern Simplified:
- Often use ready-made undergarments
- Velcro closures available
- Pre-tied obi options
```

### **Kimono Types by Formality:**

**Women's Kimono Hierarchy:**
```
Tomesode (Most Formal):
- Married women
- Black with designs below waist only
- Family crests (kamon) - 5 for maximum formality
- Weddings, important ceremonies
- Gold and silver accents

Furisode (Young Women, Very Formal):
- Unmarried women only
- Long sleeves (114cm, swinging sleeves)
- All-over patterns
- Brightest colors acceptable
- Coming-of-age ceremony (Seijin-shiki)
- Weddings (guests)

Houmongi (Semi-Formal):
- Any age, marital status
- Flowing patterns across seams
- Tea ceremonies, parties
- Sophisticated, elegant

Iromuji (Subtle Formal):
- Single color, no patterns
- Simple elegance
- Tea ceremony common
- Can be dressed up/down with accessories

Komon (Casual):
- Small repeated patterns
- Daily wear (historically)
- Shopping, casual meetings
- Wide variety of patterns

Yukata (Most Casual):
- Summer cotton kimono
- Festivals, bathhouse
- No nagajuban needed
- Simpler obi
- Seasonal summer patterns
```

**Men's Kimono:**
```
Formal:
- Black with 5 family crests
- Hakama (pleated pants/skirt over kimono)
- Haori (jacket)
- Subdued colors

Casual:
- Darker colors (navy, brown, green, gray)
- Subtle patterns
- Simpler than women's
- Shorter length
```

### **Obi Types & Tying:**

**Women's Obi:**
```
Maru Obi (Most Formal):
- Fully woven/embroidered both sides
- 30cm wide, 4m+ long
- Very expensive
- Heavy, difficult to tie

Fukuro Obi (Formal):
- Pattern on one side only
- 30cm wide, 4.2m long
- Most common formal obi
- Wedding, ceremonies

Nagoya Obi (Semi-Formal):
- Pre-sewn into shape
- Easier to tie
- Daily wear, casual occasions
- 360cm length

Hanhaba Obi (Casual):
- Half-width (15cm)
- Yukata standard
- Simple, easy tying
- Summer use
```

**Obi Musubi (Knots):**
```
Taiko Musubi (Drum):
- Most common formal
- Box-like shape at back
- Elegant, balanced

Bunko Musubi (Butterfly):
- Furisode standard
- Large, dramatic bow
- Young women

Kai-no-Kuchi (Clam Shell):
- Small, neat
- Older women
- Conservative, practical
```

```
October - May (Awase):
- Lined kimono
- Heavier fabrics
- Warmer

June, September (Hitoe):
- Unlined
- Transitional
- Medium weight

July, August (Ro, Sha):
- Sheer, gauzy

- Cooling
```

### **Decoration Techniques:**

**Dyeing Methods:**
```
Yuzen (Hand-Painted):
- Paste-resist dyeing
- Intricate, detailed designs
- Kyoto yuzen (elegant) vs Kaga yuzen (naturalistic)
- Most prestigious

Shibori (Tie-Dye):
- Various binding techniques
- Creates patterns through resist
- Kanoko (fawn spots) most prized
- Time-intensive, expensive

Katazome (Stencil Dyeing):
- Paper stencils
- Repeated patterns
- More affordable than hand-painting
- Folk art aesthetic

Ro-ketsuzome (Wax Resist):
- Similar to batik
- Wax applied, then dyed
- Can create complex designs
```

**Embellishment:**
```
Embroidery (Nui):
- Gold and silver threads
- Silk threads
- Raised, dimensional
- Formal kimono

Surihaku (Foil):
- Gold or silver leaf applied
- Glue pattern, then leaf
- Brilliant metallic effect
- Formal occasions

Kinkoma (Gold Painting):
- Gold powder mixed with glue
- Painted designs
- Luxurious effect
```

### **Wearing Rules (Kitsuke):**

**Proper Wrapping:**
```
Critical Rules:
- Always wrap left over right (right over left = death shroud)
- Collar bones should be visible
- Hem should touch top of foot
- No wrinkles or bunching
- Straight lines essential

Adjustments:
- Ohashori (fold at waist, women only)
- Creates proper length
- Men: no fold, shorter kimono
```
