# STYLE 113: TECHNICAL ILLUSTRATION

## Overview
Precise diagrams and cutaway views showing how things work, emphasizing accuracy, clarity, and educational value over artistic expression.

## Key Characteristics
- **Isometric or Orthographic Projection**: Precise angles
- **Cutaway Views**: Internal mechanism visibility
- **Labels and Annotations**: Clear text integration
- **Precise Measurements**: Technical accuracy
- **Clean Vector Lines**: Sharp, professional
- **Exploded Views**: Component separation
- **Cross-Sections**: Internal structure
- **Assembly Diagrams**: Step-by-step clarity

## Essential Keywords
`technical illustration`, `engineering diagram`, `cutaway view`, `isometric drawing`, `technical diagram`, `exploded view`, `assembly illustration`, `schematic art`

## Technical Parameters
- **Projection**: 30° isometric, 45° dimetric, orthographic
- **Line Weight**: 0.25-1.0mm precise
- **Scale**: Consistent, noted
- **Accuracy**: Dimensionally correct
- **Rendering**: Line art or subtle shading

## Best Use Cases
- User manuals, Patent drawings, Engineering docs, Assembly instructions, Educational materials, Service guides, Architecture plans, Product specifications
