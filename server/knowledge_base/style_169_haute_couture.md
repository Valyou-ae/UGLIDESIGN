# 169. HAUTE COUTURE

### **Core Characteristics:**
- **Definition**: High fashion, custom-made, hand-crafted
- **Philosophy**: Art meets fashion, wearable sculpture
- **Construction**: Hundreds of hours, master artisans
- **Exclusivity**: Limited clients, astronomical prices
- **Innovation**: Pushes boundaries, trendsetting

### **Legal Definition (Chambre Syndicale):**

**Paris Haute Couture Requirements:**
```
Official Criteria:
- Atelier in Paris
- Minimum 20 full-time employees
- Present 50+ original designs per season
- 2 collections yearly (January & July)
- Made-to-measure for private clients
- Hand-executed techniques
- Multiple fittings required

Protected Term:
- "Haute Couture" legally protected in France
- Only members of Chambre Syndicale can use term
- Guest members and correspondents exist
```

### **Construction DNA:**

**Hand Techniques:**
```
Sewing Methods:
- Hand stitching (majority of garment)
- Invisible seams
- Couture buttonholes
- Hand-sewn linings
- Bias-cut mastery
- Draping on live models

Embellishment:
- Hand embroidery
- Beading (thousands of hours)
- Feather application
- Lacework
- Appliqué
- Fabric manipulation
```

**Fitting Process:**
```
The Toile:
- Muslin prototype first
- Multiple fittings (3-5 minimum)
- Adjustments for client's body
- Perfect fit essential
- Client involvement throughout

Timeline:
- 100-400 hours per garment (typical)
- 1000+ hours for extreme pieces
- Several weeks to months
```

### **Couture Categories:**

**Day Couture:**
```
Characteristics:
- Suits (perfectly tailored)
- Day dresses
- Coats
- More wearable (relatively)
- Fine fabrics
- Impeccable tailoring

Example: Chanel tweed suit
- Hand-woven tweed
- Chain-weighted hem
- Silk lining
- Custom buttons
- Perfect shoulder line
```

**Evening Couture:**
```
Characteristics:
- Ball gowns
- Cocktail dresses
- Red carpet pieces
- Maximum embellishment
- Dramatic silhouettes
- Showstopper intent

Example: Valentino Red Dress
- Signature Valentino red
- Hand-beaded bodice
- Silk faille or taffeta
- Dramatic train
- Museum-quality construction
```

**Conceptual Couture:**
```
Characteristics:
- Runway only (not for sale)

Example: Schiaparelli Surrealist pieces
```
