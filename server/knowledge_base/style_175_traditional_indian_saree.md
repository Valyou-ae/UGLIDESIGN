# 175. TRADITIONAL INDIAN (SAREE)

### **Core Characteristics:**
- **Ancient**: 5000+ year history, Indus Valley civilization
- **Unstitched**: Single piece of fabric, 5-9 meters long
- **Versatile**: Hundreds of regional styles, draping methods
- **Living Tradition**: Daily wear for millions, evolving continuously
- **Cultural Symbol**: Grace, tradition, regional identity

### **Saree Fundamentals:**

**Basic Structure:**
```
Components:
- Main fabric: 5-9 meters (most common: 5.5-6m)
- Width: 1.1-1.4 meters (standard)
- Pallu (pallav): Decorative end piece
- Border (kinara): Along both edges
- Body (field): Main undecorated area

Traditional Set:
- Saree (main drape)
- Blouse (choli): Fitted, midriff-baring
- Petticoat (underskirt): Matches saree color
- Sometimes: Dupatta (matching scarf)
```

### **Regional Draping Styles:**

**Major Styles (100+ exist):**
```
Nivi Style (Most Common):
- Origin: Andhra Pradesh
- 6 pleats in front
- Pallu over left shoulder
- Modern standard across India
- Office-appropriate

Bengali Style:
- No pleats
- Pallu comes from back
- Draped over left shoulder, then right
- Keyhole drape
- Sindoor-daan (vermillion box) in pallu end
- Worn without petticoat traditionally

Gujarati/Seedha Pallu:
- Pallu over right shoulder (front to back)
- Front pleats
- Rajasthani variation: fuller pleats
- Head covering option

Maharashtrian Nauvari:
- 9-yard saree
- Pant-like draping (no petticoat)
- Practical for work
- Ancient style

South Indian Madisar:
- Tamil Brahmin style
- 9-yard saree
- Worn without petticoat
- Elaborate draping
- Religious ceremonies

Odia Style:
- Eastern India
- Pallu wrapped around
- Unique pleating
- Temple sculptures depict

Lehenga Style:
- North India
- Worn like lehenga (skirt)
- Pallu separate
- Bridal occasions

Butterfly/Ulta Pallu:
- Pallu pleated and pinned on shoulder
- Fan-like appearance
- Modern, dramatic
- Fashion-forward
```


### **Draping Tutorial (Nivi Style):**

```
Steps:
1. Tuck one end into petticoat (right side)
2. Wrap around once (full circle)
3. Create 6-7 pleats (5-6 inches wide)
4. Tuck pleats into petticoat (center front)
5. Drape remaining fabric around once more
6. Bring over left shoulder (pallu)
7. Pin pallu to blouse (left shoulder)
8. Adjust pleats and draping
9. Pin where needed for security

Final Check:
- Pleats facing left
- Hem touches floor (or slightly above)
- Pallu length appropriate
- No wrinkles
- Comfortable movement
```
