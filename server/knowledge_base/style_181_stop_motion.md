# 181. STOP MOTION

### **Core Characteristics:**
- **Physical**: Real puppets, sets, materials photographed
- **Frame-by-Frame**: 24 fps, tiny movements per frame
- **Tangible**: Texture, weight, handmade aesthetic
- **Imperfect**: Slight jitters, surface imperfections (charm)
- **Laborious**: Months to years for feature films

### **Puppet Construction:**

**Materials:**

**Foam Latex:**
```
Properties:
- Soft, squishy (skin-like)
- Can be painted
- Durable but delicate
Used In:
- Nightmare Before Christmas (Jack Skellington)
- Corpse Bride
- ParaNorman (faces)
```

**Silicone:**
```
Properties:
- More durable than foam latex
- Flexible, skin-like
- Translucent (subsurface scattering!)
Used In:
- Kubo and the Two Strings
- Missing Link (Laika's evolution)
```

**Replacement Animation:**
```
Technique:
- Multiple pre-made parts
- Swap between frames
- Faces most common (Laika specialty)
Laika's Innovation:
- 3D printed faces (thousands of options)
- Library of expressions
- Smooth, expressive animation
- Coraline: 207,000 face parts printed
```

### **Studio Styles:**

**Aardman Animations (UK):**
```
Materials:
- Plasticine (clay)
- Visible fingerprints (intentional)
- Warm, tactile feel
Style:
- British humor
- Exaggerated expressions
- Claymation boiling aesthetic
- Charming imperfection
Films:
- Wallace & Gromit (1989-2008)
- Chicken Run (2000)
- Shaun the Sheep
```

**Laika (USA):**
```
Materials:
- Foam latex, silicone
- 3D printed replacement faces
- High detail, intricate
Style:
- Dark, gothic fairy tales
- Cinematic lighting
- Epic scope
- Hybrid (stop motion + CG effects)
Films:
- Coraline (2009)
- ParaNorman (2012)
- Kubo and the Two Strings (2016)
```

**Wes Anderson Stop Motion:**
```
Aesthetic:
- Symmetrical compositions
- Pastel, muted color palettes
- Flat, frontal staging
- Whimsical, deadpan
Films:
- Fantastic Mr. Fox (2009)
- Isle of Dogs (2018)
```

**Tim Burton/Henry Selick:**
```
Style:
- Gothic, dark fairy tale
- Spindly character proportions
- Black and white (striped patterns)
- German Expressionism influence
Films:
- The Nightmare Before Christmas (1993)
- Corpse Bride (2005)
```
