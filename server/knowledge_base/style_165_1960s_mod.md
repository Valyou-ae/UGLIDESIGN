# 165. 1960s MOD

### **Core Characteristics:**
- **Silhouette**: Shift dress, mini skirt, geometric shapes
- **Aesthetic**: Youthful, futuristic, Pop Art influenced
- **Revolution**: Youth-driven, anti-establishment
- **Innovation**: Synthetic fabrics, bold graphics

### **Early 60s (1960-1964): Jackie Kennedy Era**

**Refined Elegance:**
```
Women:
- A-line dresses (structured)
- Pillbox hats
- Three-quarter sleeves
- Knee-length skirts
- Kitten heels
- Simple, clean lines
- Pastel colors
- Refined, ladylike

Men:
- Slim-cut suits (Continental style)
- Narrow lapels
- Skinny ties
- Chelsea boots emerging
- Ivy League look
```

### **Mid-Late 60s (1965-1969): Mod Revolution**

**The Mini Revolution:**
```
Mary Quant's Innovation:
- Mini skirt (4-6 inches above knee)
- Shift dress (straight, no waist)
- Hot pants (short shorts)
- Geometric shapes
- Bold color blocking

Mod Dress Codes:
- A-line mini dress
- Op-Art patterns
- Color blocking
- PVC plastic materials
- Go-go boots (white, knee-high)
```

**The London Mod Look:**
```
Women:
- Mini skirts/dresses
- Geometric patterns
- Bold stripes
- Color blocking
- Turtlenecks
- Vinyl/PVC accessories
- Pale lipstick
- Heavy eye makeup (Twiggy look)
- Geometric haircuts (Vidal Sassoon)

Men:
- Slim-fitting suits
- Narrow trousers
- Button-down shirts
- Parkas (with RAF roundels)
- Fred Perry polos
- Desert boots or Chelsea boots
- Mod haircuts (short, neat)
```

### **Color Revolution:**

**Early 60s Palette:**
- Pastels: pink, blue, yellow
- Classic: navy, white, beige
- Refined and conservative

**Late 60s Mod Palette:**
```
Primary Brights:
- Orange, yellow, red
- Royal blue, emerald
- Hot pink, lime green
- Purple, turquoise

Color Blocking:
- High contrast combinations
- Black and white graphics
- Neon accents
- Psychedelic influence (late 60s)
```

### **Fabric Innovation:**

**Space Age Materials:**
```
Synthetics:
- PVC (shiny, plastic)
- Vinyl (patent finish)
- Polyester (easy care)
- Acrylic (bright colors)
- Nylon (stretchy, modern)

Traditional Updated:
- Wool crepe (structured)
- Cotton piqué (textured)
- Silk (for luxury pieces)
```

**Texture & Finish:**
- High gloss, shiny
- Matte geometric prints
- Metallic fabrics
- Textured weaves
- Op-art patterns

### **The Mod Subcategories:**

**Carnaby Street Mod:**
```
Characteristics:
- Most commercial/mainstream
- Bright colors
- Union Jack motifs
- Op-art patterns
- Accessible to masses

Key Pieces:
- Mini dresses
- Geometric prints
- Platform shoes
- Bold accessories
```

**French Mod (Courrèges):**
```
Characteristics:
- Futuristic, space age
- Architectural shapes
- White/silver dominant
- Minimalist elegance
- Avant-garde

Key Pieces:
- A-line mini dresses
- White go-go boots
- Structured coats
- Helmet-style hats
- Geometric sunglasses
```

**Italian Mod:**
```
Characteristics:
- Sophisticated edge
- Sharp tailoring
- Continental elegance
- Quality fabrics
- Refined details

Key Pieces:
- Slim suits
- Fitted shirts
- Chelsea boots
- Sleek accessories
```

### **Men's Mod Evolution:**

**The Three-Button Suit:**
```
Characteristics:
- Narrow lapels
- High button stance
- Slim, tapered trousers
- No cuffs
- Vented jacket
- Precise fit

Fabrics:
- Mohair (shiny finish)
- Sharkskin (subtle sheen)
- Wool gabardine
- Two-tone weaves
```

**Mod Casual:**
```
- Polo shirts (Fred Perry, Ben Sherman)
- Harrington jackets
- Sta-Prest trousers
- Turtlenecks
- Striped boat-neck shirts
```

### **Iconic Mod Looks:**

**Twiggy (Lesley Lawson):**
```
Defining Features:
- Boyish, androgynous figure
- Geometric haircut (Vidal Sassoon)
- Heavy upper lash mascara
- Lower lash painted on
- Pale lips
- Ultra-thin build
- Mini dresses
- Flat shoes or low heels
```

**The Mod Girl:**
```
Hair:
- Geometric bobs
- Straight, sleek
- Center or side part
- Hair bands
- Sometimes wigs for volume

Makeup:
- Pale foundation
- Heavy black eyeliner
- False lashes (top and bottom)
- Pale or white lipstick
- Defined eyebrows
- Blush minimal

Nails:
- Pale pink or white
- Short, rounded
- Sometimes geometric designs
```

### **Accessories DNA:**

**Women:**
```
Footwear:

Bags:

```
